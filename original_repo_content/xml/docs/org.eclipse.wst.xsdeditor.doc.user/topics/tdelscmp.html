Deleting XML schema components

If you have created any XML schema components you no longer need, you can delete them.

The following instructions were written for the Resource perspective, but they will also work in many other perspectives.

To delete an XML schema component, complete the following steps:

  1. Open your XML schema in the XML schema editor.
  2. In the Outline view, click the item to delete.
  3. Right-click the item, and, from its pop-up menu, click Delete.

The XML schema editor has a built-in mechanism to handle referential integrity issues. When you delete certain components, cleanup will automatically occur.
