Importing DTDs

If you want to work with DTD files that you created outside of the product, you can import them into the workbench and open them in the DTD editor.

To import a DTD into the workbench, complete the following steps:

  1. Click File > Import . Select General > File System and click Next.
  2. Fill in the fields in the Import wizard as necessary.
  3. Click Finish to exit the wizard and import the files.
  4. Select the DTD file in the Navigator view and double-click it to open it in the DTD editor.

After you have imported a DTD file into the workbench and opened it in the DTD editor, you can edit it manually.
