Specifying XML default encoding line delimiters

This page lets you specify the line delimiter and the text encoding that will be used when you create or save an XML related file.

To specify the line delimiter you want to use, complete the following steps:

  1. Click Window > Preferences and select XML > XML Files.
  2. From the Encoding field, specify the default character encoding set that is used when creating XML related files (DTDs, XML files, XML schemas). Changing the encoding causes any new, XML related files that are created from scratch to use the selected encoding.
  3. Select General > Workspace in the left pane of the Preferences window and chose either Default or Other in the New text file line delimiter section. Select the operating system that applies to your development or deployment platform from the Other drop down list.
  4. Click Apply, then OK.
