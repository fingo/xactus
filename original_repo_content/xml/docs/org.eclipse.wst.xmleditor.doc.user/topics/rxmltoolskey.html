Source editor and XML tools keyboard shortcuts

The following keyboard shortcuts are available for the XML, XSD, WSDL, HTML, JSP, CSS and DTD editors.

Shortcuts available in all source editors

Command Shortcut
Go To Matching Bracket Ctrl+Shift+P
Restore Last Selection Atl+Shift+Down
Select Enclosing Element Alt+Shift+Up
Select Next Element Alt+Shift+Right
Select Previous Element Alt+Shift+Left

Shortcuts available in the XML, XSD, WSDL, HTML, JSP, and CSS editors (excludes the DTD editor)

Command Shortcut
Format Ctrl+Shift+F
Format Active Elements Ctrl+I

Shortcuts available in the XML, XSD, WSDL, HTML, and JSP editors (excludes the DTD and CSS editors)

Command Shortcut
Add Block Comment Ctrl+Shift+/
Go To Matching Tag Ctrl+Shift+>
Go To Next Sibling Ctrl+Shift+Down
Occurrences in File Ctrl+Shift+A
Open Selection F3
Go To Previous Sibling Ctrl+Shift+Up
Remove Block Comment Ctrl+Shift+\
Toggle Comment Ctrl+Shift+C

Shortcuts available in the design view of the XSD and WSDL editors

Command Shortcut
Zoom In Ctrl+=
Zoom Out Ctrl+-

Command Shortcut
Export Diagram as Image Ctrl+Shift+X
