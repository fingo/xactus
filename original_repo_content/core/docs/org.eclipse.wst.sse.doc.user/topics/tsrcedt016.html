Deleting or hiding snippet items or drawers

This documentation describes how to delete or hide drawers and items in the Snippets view.

You can hide any drawer or item that shows up in the Snippets view. If the drawer or item is user-defined, you can delete it permanently. To delete or hide a drawer or item, complete the following steps:
  1. Select the item or drawer that you want to remove.
  2. Right-click and select Customize from the pop-up menu.
  3. Select the item or drawer and click Delete or Hide.
  4. Click OK. The item or drawer will be deleted or hidden from the Snippets view.
