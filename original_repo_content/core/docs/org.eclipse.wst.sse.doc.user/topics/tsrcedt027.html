Adding and removing markup language templates - overview

Content assist provides templates, or chunks of predefined code, that you can insert into a file. You can use the default templates as provided, customize the default templates, or create your own templates.

Templates are available for the following markup languages:
  • HTML or XHTML
  • JSP
