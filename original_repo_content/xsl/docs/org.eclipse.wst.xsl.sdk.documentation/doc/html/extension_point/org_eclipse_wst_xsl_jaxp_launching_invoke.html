XSLT Processor Invoker

This extension point is internal

Identifier:
org.eclipse.wst.xsl.launching.jaxp.invoke

Description:
This extension point is internal and should not be used by any other plugins.

Configuration Markup:

<!ELEMENT extension (invoker)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT invoker EMPTY>

<!ATTLIST invoker

id        CDATA #REQUIRED

class     CDATA #REQUIRED

classpath CDATA #REQUIRED

>

  • id - Unique id for this invoker
  • class - A class that implements the IProcessorInvoker interface
  • classpath - A semi-colon separated list of jars relative to the root of the contributing plugin. One of the jars must contain the Invoker class named in the class attribute.
